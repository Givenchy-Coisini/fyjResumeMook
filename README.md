## Electron + React Hooks + TS 实战开发：从 0 到 1 实现简历平台

### 开发

> 由于网络问题，往往安装 Electron 会很慢，此时可以考虑换个淘宝源

```js
// 1.下载
npm run install
// 2. 运行
npm run start:render
npm run start:main
```
